How do I start chopping?

If you are using woodcoind you need to issue the command woodcoind setgenerate true. If you are using woodcoin-qt, you need to navigate as follows:

  • Help ==> Debug Window ==> Console Tab.
  • Then in the console type the command setgenerate true and press Enter.
  • To check your woodcutting progress, go to the same console and type getmininginfo.

I'm behind a checkpoint and alerts aren't working. What’s up?

The philosophy of woodcoin is complete decentralization. Checkpoints and alerts are not used.

How can I contribute?

Start chopping wood! By chopping wood and collecting LOGs, you are helping secure the Woodcoin network. By using LOGs as tokens of exchange, you are helping to spread the word about woodcoin.

The code which runs woodcoin will always be open source. Take a look and submit your changes on github. There are many other tools still missing from the woodcutting arsenal and new ideas of all sorts are appreciated. If you can’t code it up yourself, tell us about it on one of the forums and maybe we can do it for you.
