The following are Woodcoin wallets, nodes, and tools tested and built by the woodcoin.org team. Search for them or find them in the github repo below

  • Woodcoin-QT
  • Original graphical wallet.
  • woodcoind
  • Original headless client for servers and non-graphical users.
  • fo-realz-woodcoin
  • Satoshi 0.5.3 based command line full node and reference client.
  • woodcoin-minimal
  • A modern command line client with minimal LOC
  • Vanitygen for Woodcoin
  • Allows advanced dwarves to create custom log addresses for daily use.
  • GPU woodcutter (nvidia)
  • GPU double skein miner.
  • GPU woodcutter (amd)
  • Double skein miner for AMD GPUs(under development)
  • Offline Browser paper wallet tool
  • Essential tool for paper wallets and address manipulation

WINDOWS and MAC users:

Proprietary software systems are almost always a bad idea.

They are less reliable, harder to use, and they will not continue to be supported as the controlling interest requires upgrades or disappears.

They also cannot possibly be secure.
